HOW TO USE

사용 방법

  1. 목표 이름, 시작일, D-day를 설정합니다.
  2. 매일 완료한 날짜에 X 표시 버튼을 눌러 기록합니다.
  3. 각 날짜 카드의 D-숫자 / D+숫자로 남은 기간을 직관적으로 확인합니다.
  4. 선택한 날짜 영역에서 상태를 다시 확인하거나 취소할 수 있습니다.
